    Transformer Item¶

    Type: object Additional Properties Allowed: True

    Property Type Required Format Title
    kva integer Capacity in kVA
    count integer Transformer Count
    is_regulator boolean Is Regulator
    is_substation_transformer boolean Is Substation Transformer
    num_phase integer Number of Phase
    high_kv number High Voltage kV
    low_kv number Low Voltage kV
    avg_customers_served number Average Customers Served
    min_customers_served number Minimum Customers Served
    max_customers_served number Maximum Customers Served
    std_customers_served number \| string Standard Deviation Customers Served
